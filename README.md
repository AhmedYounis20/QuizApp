# Quiz App

The **Quiz App** is an engaging web application developed using ASP.NET for the API and React for the front-end. üöÄ It allows users to create quizzes and participants to log in using their email and name, answering quizzes with randomly generated questions, and receiving scores stored within the system.

## Key Features:

- **Quiz Creation:** Users can create quizzes with diverse and customizable questions.
- **User Authentication:** Participants log in with their email and name to answer quizzes.
- **Randomized Questions:** Each participant gets a unique set of questions for a personalized experience.
- **Score Tracking:** Scores are calculated and saved within the system for review.

## System Capabilities:

- **Dynamic Quiz Interface:** Enjoy a dynamic interface for creating and answering quizzes.
- **User-Friendly Authentication:** Easily log in using email and name credentials.
- **Personalized Quizzes:** Participants receive a unique set of questions for an engaging experience.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ahmedyounis20/sobamobile.git
   ```
2. Open QuizApi folder
3. Run Api with cli or with visual studio
   ```sh
   dotnet run 
   ```
4. Open quiz-client folder 
5. Install NPM Packages
   ```sh
    npm install
   ```
6. Run client
   ```sh
   npm start
   ```
